How to setup Andorid Studio

    • Please prefer video tutorials how to install Andorid Studio
    • Please prefer video tutorials how to install Mac
  1. Head over to this link to get the Android Studio executable or zip file.
  2. Click on the Download Android Studio Button.


  3. Click on the “I have read and agree with the above terms and conditions” checkbox followed by the download button.




    4. Click on the Save file button in the appeared prompt box and the file will start downloading.


  4. After the downloading has finished, open the file from downloads and run it. It will prompt the following dialog box.


    5. Click on next. In the next prompt, it’ll ask for a path for installation. Choose a path and hit next.


  5. It will start the installation, and once it is completed, it will be like the image shown below.



    6. Click on next.




  6. Once “Finish” is clicked, it will ask whether the previous settings need to be imported [if the android studio had been installed earlier], or not. It is better to choose the ‘Don’t import Settings option’.


  7. Click the OK button.
  8. This will start the Android Studio.




    9. Meanwhile, it will be finding the available SDK components.



  9. After it has found the SDK components, it will redirect to the Welcome dialog box.



  10. Click on Next.



    11. Choose Standard and click on Next. Now choose the theme, whether the Light theme or the Dark one. The light one is called the IntelliJ theme whereas the dark theme is called Dracula. Choose as required.




  11. Click on the Next button.

  12. Now it is time to download the SDK components.



  13. Click on Finish. Components begin to download let it complete.



    14. The Android Studio has been successfully configured. Now it’s time to launch and build apps. Click on the Finish button to launch it.


  14. Click on Start a new Android Studio project to build a new app.



    15. To run your first android app in Android Studio you may refer to Running your first Android app.

How to change App Version

  1. go to pubspec.yaml
  2. EX.Update version:A.B.C+X in pubspec.yaml.
  3. For Android: A.B.C represents the versionName such as 1.0.0. X (the number after the +) represents the versionCode such as 1, 2, 3, etc.
  4. Do not forget to execute flutter packages get, flutter build or flutter run after this step
  5. For iOS
  6. A.B.C represents the CFBundleShortVersionString such as 1.0.0. X (the number after the +) represents the CFBundleVersion such as 1, 2, 3, etc.
  7. Do not forget to execute flutter packages get, flutter build or flutter run after this step
