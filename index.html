How to Setup Java jdk-11 on windows

  1. If your System already have Java jdk-8 installed then you can skip this step.
  2. Download Java-11 jdk by clicking below link.
    (JAVA jdk-11)
  3. Download Java-11 JDK for your version 32 bit or JDK download 64 bit.



  4. When you click on the Installation link the popup will be open. Click on I reviewed and accept the Oracle Technology Network License Agreement for Oracle Java SE development kit and you will be redirected to the login page. If you don’t have an oracle account you can easily sign up by adding basics details of yours.




  5. Once the Java JDK 11 download is complete, run the exe for install JDK. Click Next



  6. Select the PATH to install Java in Windows… You can leave it Default. Click next.



  7. Once you install Java in windows, click Close



  8. Now set your enviournment variable
  9. From the Start search bar in , type ‘env’ and select Edit environment variables for your account



  10. Under User variables check if there is an entry called Path:
  11. Click on Edit. then Dialog will open from it, now click on new and Copy the path of bin folder which is installed in JDK folder. then Paste Path of bin folder in Variable value. Click on OK Button.


  12. Repeat above process but this time set variable name JAVA_HOME instead of "path/PATH", and set JDK bin folder path in the variable value.
  13. Now Restart your pc for changes to take effect

  14. Now check everything is ok or not
  15. Go to command prompt and type javac commands.
    If you see a screen like below, Java is installed.

How to setup Backend

  1. Login to your WordPress Dashboard.
  2. In your WordPress Admin Menu, go to Plugins > Add New.
  3. Click on Upload Plugin button found on top left corner of page.
  4. Click on Browse (1), Select the .zip file of your plugin in your computer, and click Install Now (2) button.
  5. At this point, the plugin is installed. You can click on Activate Plugin link to work with it.

How to link Application with Backend

  1. Once you've installed plugin and activated
  2. Open Flutter Project in android studio.
  3. Navigate to lib > data_provider > api_constant.dart file
  4. Add your wordpress domain in highlighted area.
  5. Run your project by connecting physical device or virtual device.
